  This is a very basic 'Restaurant' made using the Python Django Web Development package. 

  Using the admin site, you can add items, with names, descriptions, price and a URL for the image; and depending on what type of food it is, it also automatically loads the symbol for vegetarians, non-vegetarians, or vegans.

  Add it to the cart, and see your order and the total price of it, loaded using Bootstrap styling to assure a simple yet interesting layout.
